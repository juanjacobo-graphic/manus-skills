# Checklist de Evaluación de Proyectos de Inversión

*Este checklist se usa para evaluar la viabilidad, el impacto y el potencial de un plan de negocio antes de presentarlo a un comité de inversión.*

---

## Criterios de Evaluación

| Criterio | Pregunta Clave | Cumple (Sí/No/Parcial) | Observaciones |
| :--- | :--- | :--- | :--- |
| **1. Viabilidad del Mercado** | ¿Existe un mercado real, medible y accesible para el producto/servicio? | | |
| | ¿El problema que se resuelve es relevante para el público objetivo? | | |
| | ¿La ventaja competitiva es clara y sostenible en el tiempo? | | |
| **2. Viabilidad Técnica** | ¿El plan de operaciones es realista y está bien definido? | | |
| | ¿Se cuenta con los recursos (tecnología, equipo, conocimiento) para ejecutarlo? | | |
| | ¿El producto o servicio es factible de desarrollar y escalar? | | |
| **3. Viabilidad Financiera** | ¿Las proyecciones financieras son coherentes, realistas y están bien fundamentadas? | | |
| | ¿El punto de equilibrio es alcanzable en un plazo razonable? | | |
| | ¿El uso de los fondos solicitados está justificado y es eficiente? | | |
| **4. Capacidad del Equipo** | ¿El equipo emprendedor tiene la experiencia y las habilidades necesarias? | | |
| | ¿Los roles y responsabilidades están claramente definidos? | | |
| | ¿El equipo demuestra compromiso y resiliencia? | | |
| **5. Innovación** | ¿El proyecto introduce un nuevo producto, servicio, proceso o modelo de negocio? | | |
| | ¿Aporta una solución significativamente mejor a las existentes? | | |
| **6. Impacto Económico y Social**| ¿El proyecto tiene potencial para generar empleos de calidad? | | |
| | ¿Contribuye al desarrollo económico de su región/comunidad? | | |
| | ¿Genera un impacto social o ambiental positivo y medible? | | |
| | ¿Considera a población vulnerable o se ubica en zonas de interés estratégico (rural, PDET)? | | |
