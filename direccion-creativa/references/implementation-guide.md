_Guía para planificar la producción de un proyecto creativo._

# Guía de Implementación

## 1. Desglose de Tareas (Work Breakdown Structure)

- **Fases del Proyecto:** Dividir el proyecto en fases (pre-producción, producción, post-producción).
- **Tareas por Fase:** Listar todas las tareas específicas que se deben realizar en cada fase.
- **Dependencias:** Identificar qué tareas dependen de otras para poder comenzar.

## 2. Asignación de Recursos

- **Equipo:** Definir los roles y responsabilidades de cada miembro del equipo.
- **Equipamiento:** Listar el equipo técnico necesario (cámaras, luces, software, etc.).
- **Locaciones:** Gestionar la búsqueda y el permiso de las locaciones de rodaje.

## 3. Cronograma

- **Diagrama de Gantt:** Crear un cronograma visual que muestre la duración de cada tarea y sus dependencias.
- **Hitos (Milestones):** Establecer fechas clave para la entrega de avances y la revisión del proyecto.

## 4. Presupuesto Detallado

- **Costos de Personal:** Salarios del equipo.
- **Costos de Equipamiento:** Alquiler o compra de equipo.
- **Costos de Producción:** Locaciones, catering, transporte, etc.
- **Costos de Post-producción:** Edición, música, efectos visuales, etc.
- **Contingencia:** Asignar un porcentaje (generalmente 10-15%) para imprevistos.

## 5. Plan de Riesgos

- **Identificar Riesgos:** ¿Qué podría salir mal? (e.g., mal clima, problemas técnicos, retrasos).
- **Plan de Mitigación:** ¿Cómo se pueden prevenir o minimizar esos riesgos?
- **Plan de Contingencia:** ¿Qué se hará si los riesgos se materializan?
