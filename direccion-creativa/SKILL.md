# Habilidad de Dirección Creativa

Esta habilidad te guía en el proceso de dirección creativa para proyectos de publicidad y cine en Colombia.

## Flujo de Trabajo

Sigue estos pasos para desarrollar tu proyecto:

1.  **Briefing**: Define el proyecto. Usa `references/brief-template.md`.
2.  **Investigación**: Analiza el mercado. Consulta `references/market-research-guide.md`.
3.  **Generación de Conceptos**: Crea ideas. Revisa `references/concept-generation-framework.md`.
4.  **Feedback y Refinamiento**: Itera tus ideas. Usa `references/feedback-structure.md`.
5.  **Plan de Implementación**: Planifica la producción. Sigue `references/implementation-guide.md`.

## Recursos

-   **Scripts**: Automatiza tareas con los scripts en la carpeta `scripts/`.
-   **Templates**: Usa las plantillas en `templates/` para tus documentos.
